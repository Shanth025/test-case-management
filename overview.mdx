The **Test Case Management (TCM)** tool is an end-to-end platform that streamlines testing operations across the software development lifecycle — from initial planning and stakeholder approvals through execution and final sign-off.

<CardGroup cols={2}>
  <Card title="Test Plan Management" icon="list-check">
    Build and manage test plans, track feature workflow status, and drive approvals from draft to sign-off.
  </Card>
  <Card title="Automation Coverage" icon="robot">
    Monitor automated, manual, and disabled test cases across features with real-time coverage metrics.
  </Card>
  <Card title="Workflow & Approvals" icon="diagram-project">
    Move features through a structured approval lifecycle with email and in-app notifications at every step.
  </Card>
  <Card title="KPI Dashboard" icon="chart-line">
    Track sprint-level automation trends, team performance, and bottlenecks across testing cycles.
  </Card>
</CardGroup>

## Key features

<AccordionGroup>
  <Accordion title="Feature Management" icon="layer-group">
    Create features, assign approvers, link test cases, and track each feature through its full workflow lifecycle from draft to sign-off.
  </Accordion>
  <Accordion title="Test Case Management" icon="clipboard-check">
    Create, update, search, filter, and delete test cases. Supports bulk operations, advanced filtering, full version history, revert, and admin-only restore.
  </Accordion>
  <Accordion title="Import from Google Sheets" icon="file-import">
    Bulk import test cases from a Google Sheet by mapping row ranges to features. Import history is available for review.
  </Accordion>
  <Accordion title="Notifications" icon="bell">
    In-app and email notifications alert users to pending actions. View unread items, mark them as read, and review notification history.
  </Accordion>
  <Accordion title="Comments and Collaboration" icon="comments">
    Add threaded comments to features and test cases, mention teammates using `@mention`, and add emoji reactions.
  </Accordion>
  <Accordion title="User and Role Management" icon="users-gear">
    Admins add users, update roles, and manage access. The tool supports two roles: `USER` and `ADMIN`.
  </Accordion>
</AccordionGroup>
